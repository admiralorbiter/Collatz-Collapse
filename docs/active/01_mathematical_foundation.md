# Mathematical Foundations of the Collatz Research Workbench

## 1. Collatz Maps & Representations

### 1.1 Ordinary Collatz Map
For a positive integer $n \in \mathbb{N}^+$:
$$T(n) = \begin{cases} n / 2, & n \equiv 0 \pmod 2, \\ 3n + 1, & n \equiv 1 \pmod 2. \end{cases}$$

This ordinary representation is used for direct trajectory validation, peak value calculation, and baseline comparison with standard computational benchmarks.

### 1.2 Odd-Only Collatz Map
For an odd positive integer $n \in 2\mathbb{N}+1$, define the 2-adic valuation $v_2(x)$ as the largest exponent $a$ such that $2^a \mid x$. The accelerated odd-only map $S(n)$ jumps directly to the next odd integer:
$$a = v_2(3n + 1), \qquad S(n) = \frac{3n + 1}{2^a}$$

The odd-only map $S$ serves as the primary mathematical domain for symbolic search, valuation prefixes, and affine trajectory analysis.

## 2. Valuation Words & Affine Recurrences

### 2.1 Valuation Words
For an odd trajectory starting at $n_0$, record the sequence of 2-adic valuations:
$$a_i = v_2(3n_i + 1), \qquad i = 0, 1, \ldots, k-1$$

A finite sequence $(a_0, a_1, \ldots, a_{k-1})$ with $a_i \ge 1$ is a **valuation word** of length $k$. Define the partial sum of valuations:
$$A_0 = 0, \qquad A_k = \sum_{i=0}^{k-1} a_i$$

### 2.2 Exact Affine Transformation
After $k$ odd steps, the $k$-th odd term $n_k$ is an exact affine function of the starting value $n_0$:
$$n_k = \frac{3^k n_0 + c_k}{2^{A_k}}$$

where the additive constant $c_k$ is generated by the recurrence:
$$c_0 = 0, \qquad c_{i+1} = 3c_i + 2^{A_i} \quad \text{for } 0 \le i < k$$

In expanded polynomial form:
$$c_k = \sum_{i=0}^{k-1} 3^{k-1-i} 2^{A_i}$$

## 3. Closed-Form Modular Inversion Identity

### 3.1 Closed-Form Congruence Derivation & Valuation Semantics
Every valuation word $(a_0, \ldots, a_{k-1})$ of total valuation $A_k$ imposes modular congruences on the starting value $n_0$. Crucially, there are two distinct semantics:

#### 1. Broad Valuation Class (`TerminalAtLeast`)
Taking $3^k n_0 + c_k \equiv 0 \pmod{2^{A_k}}$ yields:
$$n_0 \equiv -c_k \cdot (3^k)^{-1} \pmod{2^{A_k}}$$
This congruence forces the first $k-1$ valuations $(a_0, \ldots, a_{k-2})$ exactly, and guarantees that the $k$-th valuation is **at least** $a_{k-1}$ ($v_2(3n_{k-1}+1) \ge a_{k-1}$).

#### 2. Complete Exact Valuation Cylinder (`ExactWord`)
To force the final value after $k$ divisions to be odd—and therefore obtain the exact word $(a_0, \ldots, a_{k-1})$—we apply a 1-bit lift requiring $3^k n_0 + c_k \equiv 2^{A_k} \pmod{2^{A_k + 1}}$:
$$n_0 \equiv (2^{A_k} - c_k) \cdot (3^k)^{-1} \pmod{2^{A_k + 1}}$$

## 4. Growth Debt & Exact Descent Thresholds

### 4.1 Growth Debt
The multiplicative growth ratio after $k$ odd steps is $\frac{3^k}{2^{A_k}}$. We define the **growth debt** $D_k$ as:
$$D_k = k \log_2 3 - A_k$$

### 4.2 Exact Integer Descent Threshold $B$
Suppose a valuation prefix of length $k$ satisfies $2^{A_k} > 3^k$ (multiplicative contraction). Then $n_k < n_0$ if and only if:
$$n_0 \ge B = \left\lfloor \frac{c_k}{2^{A_k} - 3^k} \right\rfloor + 1$$

## 6. Discrete Lift-Digit Decomposition & Realization Equivalence

### 6.1 Lift-Digit Decomposition Formula
Let $H_0 = 5, r_0 = 7$, and $H_m = 5 + \sum_{i=0}^{m-1} B(w_i)$.
Because prefix cylinders are nested ($r_{m+1} \equiv r_m \pmod{2^{H_m}}$), each step decomposes into:
$$r_{m+1} = r_m + d_m 2^{H_m}, \qquad 0 \le d_m < 2^{B(w_m)}$$

Cumulative summation formula:
$$r_m = 7 + \sum_{i=0}^{m-1} d_i 2^{H_i}$$

Because $d_i \ge 0$, the representative sequence $(r_m)_{m \ge 0}$ is **monotone nondecreasing**.

### 6.2 The Realization Equivalence Theorem
For an infinite itinerary $\omega = (w_0, w_1, \dots)$ with 2-adic limit point $\kappa(\omega)$:
$$\kappa(\omega) \in \mathbb{N}^+ \iff (r_m)_{m \ge 0} \text{ is bounded} \iff (r_m)_{m \ge 0} \text{ eventually constant} \iff \mathbf{d_m = 0 \text{ for all large } m}$$

### 6.3 Arithmetic No-Escape Reduction
The global survivor-set integer intersection theorem is reduced to proving:
$$X_\infty \cap \mathbb{N}^+ = \emptyset \iff \forall \omega \in L_{\text{valid}}^\mathbb{N}, \quad \neg \text{EventuallyZeroLift}(\omega)$$
